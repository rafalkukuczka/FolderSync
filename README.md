# FolderSync 
Folder synchronization. (.net 9.0 command line example project).

## Usage: FolderSync.exe source destination
- The source folder - pattern
- The destination folder - mirror of the source folder

### Example use: (please, use with **_caution!_**)
FolderSync.exe c:\temp\src c:\temp\dst
